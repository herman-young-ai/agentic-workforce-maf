## Layer dependencies

```
AppHost     -> Api, Worker                (Aspire orchestration)
Api         -> Domain, Infrastructure     (BFF — auth, middleware, endpoints)
Worker      -> Domain, Infrastructure, Agents (Durable Task, agent execution)
Agents      -> Domain                     (MAF agent wrappers, tools)
Infrastructure -> Domain                  (EF Core, Redis, Azure SDK)
Domain      -> (no dependencies)          (entities, interfaces, exceptions)
```
